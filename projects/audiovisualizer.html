Audio Visualizer

This audio visualizer has only a few song selections to choose from, but it analyzes the frequency data each frame using an analyser node. This data is used to influence many of the properties of the shapes and objects displayed on the canvas.

Created for a class in the spring of 2016, this was one of my first javascript projects. I worked on all of the code for the rectangles, center wave lines, and moving dots as well as adjusting many of the values to balance the visuals.
